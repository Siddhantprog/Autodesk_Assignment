# Assignment 1 - Reverse Words

Reverses each word in a string while keeping spaces and punctuation unchanged.

## Approach

- Traverse the string.
- Identify alphanumeric words using `isalnum()`.
- Reverse each word using `reverse()`.
- Leave spaces and punctuation unchanged.

## Complexity

- Time Complexity: O(n)
- Space Complexity: O(n)
